# MyCrate
MyCrate is a CRUD app made via the MERN stack. It allows users to catalog a digital version of either albums they own, or just enjoy. 

# Technologies Used
* MongoDB
* Express.js
* React.js
* Node.js
* Material UI
* JSX
* CSS
* Netlify
* Heroku
* Figma
* Trello
* Invision App

# Getting Started 
Follow this Link to load the app. 
https://mycrate.netlify.app/

# Future Enhancements
